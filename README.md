🚀 Características del Proyecto:

Este portafolio web fue desarrollado con tecnologías modernas y presenta un diseño profesional, limpio y totalmente responsive para mejor navegacion entre varios dispositivos

🎯 Diseño comodo:

Adaptado para varios dispositivos

🛠️ Tecnologías Utilizadas: 

Frontend: 

- HTML5: Estructura semántica y accesible
  
- CSS3: Estilos modernos y llamativos
  
- Bootstrap: Framework responsive para mejorar la navegacion y la facilidad de hacerlo sin mucho CSS3
